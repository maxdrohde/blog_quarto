11.1 Anonymous function syntax

Standard purrr syntax

map_dbl(1:10, ~.x^2)
 [1]   1   4   9  16  25  36  49  64  81 100

New R anonymous function syntax

map_dbl(1:10, \(x) x^2)
 [1]   1   4   9  16  25  36  49  64  81 100

Another example:

f <- \(x,y,z) x+y+z

f(1,2,3)
[1] 6

Even shorter:

(\(x,y,z) x+y+z)(1,2,3)
[1] 6

## Anonymous function syntax

Standard `purrr` syntax

```{r}
map_dbl(1:10, ~.x^2)
```

New R anonymous function syntax

```{r}
map_dbl(1:10, \(x) x^2)
```

Another example:

```{r}
f <- \(x,y,z) x+y+z

f(1,2,3)
```

Even shorter:

```{r}
(\(x,y,z) x+y+z)(1,2,3)
```
